# NEXO MARKET TURBO STOCK MANAGER

Remplace dans Replit :
- bot.js
- package.json

Garde ton .env actuel.

Puis dans Shell :
npm install
npm start

Ce qui change :
- Le salon stock devient une carte par offre.
- Quand tu modifies Tech 16, le bot édite seulement la carte Tech 16.
- Il ne supprime plus tout le salon.
- Telegram répond immédiatement, donc moins de chargement infini.
- Le bot crée un fichier stock-cache.json pour retenir les IDs des messages Discord.

Dans Telegram :
/start
Puis clique "🔄 Créer / réparer stock" une fois.
Ensuite utilise "📦 Modifier stock".

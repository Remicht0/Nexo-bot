# NEXO MARKET TURBO + BUMP REMINDER

Remplace dans Replit :
- bot.js
- package.json

Garde ton .env actuel.

Puis dans Shell :
npm install
npm start

Nouveau bouton Telegram :
🔔 Rappels bump Disboard

Fonctions :
- ▶️ Activer rappels : programme un rappel dans 2h
- ✅ Bump fait : programme le prochain rappel dans 2h
- ⏰ Me rappeler dans 15 min : snooze
- ⏸️ Pause rappels : coupe les rappels
- 🧪 Test 10 secondes : vérifier que ça marche

Important :
Le bot ne bump pas automatiquement. Il te rappelle juste de faire /bump manuellement, ce qui est plus safe.

# NEXO MARKET vFinal Upgrade

Remplace dans Replit :
- bot.js
- package.json

Garde ton `.env` actuel.

Puis dans le Shell :

npm install
npm start

Dans Telegram :
/start

Nouveaux menus :
- 📦 Stock
- 💸 Prix
- 🎟️ Tickets
- 👤 Clients
- 📣 Pub
- 🔔 Bump
- 🚨 Sécurité lock/unlock
- 📊 Stats

À faire juste après :
1. Telegram > /start
2. Clique 🔄 Créer / réparer stock
3. Teste 📦 Stock > KFC > statut
4. Teste 📣 Pub > Message seul à copier
5. Teste 🚨 Sécurité > Lock puis Unlock

Le bot crée `nexo-data.json` pour sauvegarder stats, tickets et clients.
Ne le supprime pas si tu veux garder l'historique.
